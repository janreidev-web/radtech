## Overview
radtech is an all-in-one developer toolkit for building interactive, modern web applications with React, Tailwind CSS, and 3D content. It offers streamlined setup, deployment, and development workflows tailored for scalable, high-performance projects.

**Why radtech?**

This project simplifies the process of creating engaging web experiences, combining optimized tooling with a modular architecture. The core features include:

- 🧩 🎯 **Deployment & Build**: Configured for efficient static asset delivery and API routing on Vercel.
- 🚀 ⚡ **Fast Development**: Integrates React and Tailwind CSS via Vite for rapid iteration and styling.
- 🖼️ 🎨 **3D Content**: Includes components like ModelLoader for immersive 3D visualization.
- 🔍 🛡️ **Code Quality**: ESLint setup enforces best practices and consistency.
- 🧱 🔧 **Modular Architecture**: Reusable components like Header, Footer, and Icon streamline development.
- 🌐 🌟 **Interactive UI**: Designed for scalable, engaging user interfaces with dynamic routing and theming.

## Getting Started
### Prerequisites
This project requires the following dependencies:
- **Programming Language:** JavaScript
- **Package Manager:** Npm

### Installation
Build radtech from the source and install dependencies:

1.  **Clone the repository:**
    ```sh
    ❯ git clone [https://github.com/janreidev-web/radtech](https://github.com/janreidev-web/radtech)
    ```
2.  **Navigate to the project directory:**
    ```sh
    ❯ cd radtech
    ```
3.  **Install the dependencies:**

    **Using [npm](https://www.npmjs.com/):**
    ```sh
    ❯ npm install
    ```

### Usage
Run the project with:

**Using [npm](https://www.npmjs.com/):**
```sh
npm run dev
